# Sample_Size
Stata code for sample size estimates

## icc_sampsi.ado
Calculates required sample size to estimate an intraclass correlation coefficient (ICC) with desired precision. Options for repetitions and confidence interval coverage.  Based on Bonett, D. G. 2002. Sample size requirements for estimating intraclass correlations with desired precision. *Statistics in Medicine* 21: 1331-1335.
